## What is CotizaClima?

CotizaClima streamlines the entire quotation workflow for HVAC businesses:

- **Multi-step wizard** guides users through equipment selection, service options, and client details
- **Equipment catalog** with advanced filtering by brand, BTU capacity, and refrigerant type
- **Client management** with RUT-based deduplication and complete contact history
- **Professional PDFs** generated on-demand with your branding and pricing
- **Firebase integration** for authentication, real-time data sync, and persistence
- **Mobile-first design** that works seamlessly on phones, tablets, and desktops

## Built for Chilean Market

CotizaClima is purpose-built for HVAC businesses operating in Chile:

- **RUT validation** for client identification and deduplication
- **Chilean regions** pre-configured in address forms
- **Currency formatting** in Chilean Pesos (CLP)
- **Spanish UI** with localized labels and messages
- **GPS integration** for address autocomplete using Chilean geocoding

## Technical Stack

CotizaClima is a client-side web application with Firebase backend:

- **Frontend**: Vanilla JavaScript with responsive CSS
- **Authentication**: Firebase Authentication with email/password
- **Database**: Firebase Realtime Database for quotes, clients, and equipment catalog
- **PDF Generation**: jsPDF with auto-table plugin
- **Deployment**: Static hosting (works with any web server)
