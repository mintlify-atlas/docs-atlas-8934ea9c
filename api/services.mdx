## Overview

The Services module defines available installation and maintenance services, along with PDF content templates for quote generation.

## SERVICES Array

Array of service objects available in the application. Defined in `app.js:5-12`.

<ResponseField name="SERVICES" type="array">
  Array of 6 service objects:
</ResponseField>

### Service Object Structure

<ParamField path="id" type="string">
  Service identifier: `'instalacion'`, `'mantencion'`, `'cambio'`, `'reparacion'`, `'garantia'`, `'presupuesto'`
</ParamField>

<ParamField path="name" type="string">
  Display name (Spanish)
</ParamField>

<ParamField path="desc" type="string">
  Service description
</ParamField>

<ParamField path="price" type="number">
  Default price in CLP (Chilean Pesos)
</ParamField>

<ParamField path="icon" type="string">
  Font Awesome icon class (e.g., `'fa-hammer'`)
</ParamField>

## SERVICE_PDF_CONTENT Object

PDF content templates for generating detailed quote PDFs. Each service type has custom text blocks. Defined in `app.js:14-57`.

<ResponseField name="SERVICE_PDF_CONTENT" type="object">
  Object with service IDs as keys:
</ResponseField>

### Content Block Structure

<ParamField path="title" type="string">
  PDF section title
</ParamField>

<ParamField path="relevant" type="array">
  Array of relevant information paragraphs
</ParamField>

<ParamField path="extra" type="array">
  Array of additional information bullet points
</ParamField>

### Available Content Blocks

The object contains detailed PDF content for:
- `instalacion` - Installation service information
- `mantencion` - Maintenance service information  
- `reparacion` - Repair/diagnostic service information

<Note>
  The `cambio` service ID maps to `instalacion` content in PDF generation.
</Note>

<CodeGroup>
```javascript Content Structure
const SERVICE_PDF_CONTENT = {
  instalacion: {
    title: 'Instalación de Aire Acondicionado',
    relevant: [
      'Agradecemos la confianza depositada en RefriPro.',
      'La presente cotización considera la instalación de equipo de aire acondicionado tipo Split, realizada bajo criterios técnicos y buenas prácticas de climatización para asegurar un funcionamiento seguro y eficiente.',
      'El servicio incluye instalación de unidad interior y exterior, conexión de cañerías de refrigeración, conexión eléctrica básica, drenaje de condensado y puesta en marcha del equipo, verificando su correcto funcionamiento.'
    ],
    extra: [
      'La instalación considera condiciones estándar, incluyendo hasta 3 metros de interconexión (cañerías, cableado y drenaje) entre unidades.',
      'Trabajos adicionales como mayor distancia de cañería, canaletas o soportes especiales serán informados y cotizados previamente.',
      'El cliente debe contar con punto eléctrico operativo y acceso adecuado al lugar de instalación.',
      'La instalación cuenta con 1 año de garantía por el trabajo realizado por RefriPro.'
    ]
  },
  // ... mantencion and reparacion blocks
};
```

```javascript Usage Example
// Get installation PDF content
const installContent = SERVICE_PDF_CONTENT.instalacion;
console.log(installContent.title); // 'Instalación de Aire Acondicionado'

// Access relevant info paragraphs
installContent.relevant.forEach(paragraph => {
  console.log(paragraph);
});

// Access extra info bullets
installContent.extra.forEach(item => {
  console.log('- ' + item);
});
```
</CodeGroup>

### PDF Generation

These content blocks are automatically included in generated PDF quotes when the corresponding service is selected.

<CodeGroup>
```javascript Integration Example
// Used by appendServiceInfoToPdf function
function getServiceInfoKeys(selectedServiceItems) {
  const keys = new Set();
  selectedServiceItems.forEach(item => {
    const id = item && item.svc ? item.svc.id : '';
    if (!id) return;
    
    // 'cambio' maps to 'instalacion' content
    if (id === 'cambio') {
      keys.add('instalacion');
      return;
    }
    
    if (SERVICE_PDF_CONTENT[id]) {
      keys.add(id);
    }
  });
  return [...keys];
}
```
</CodeGroup>
