CotizaClima generates professional PDF quotations using the jsPDF library with autoTable plugin, producing polished documents ready for client delivery.

## Overview

The PDF generation system creates comprehensive quotation documents that include:

- Branded header with logo and company information
- Client details and installation address
- Itemized equipment, services, and accessories
- Pricing breakdown with discount support
- Service-specific terms and conditions
- Professional footer with quote number

#### Rendering Service Info (app.js:1451-1503)

```javascript
function appendServiceInfoToPdf(doc, startY, margin, pageW, selectedServiceItems) {
  const keys = getServiceInfoKeys(selectedServiceItems);
  if (!keys.length) return startY;
  
  let y = startY;
  
  keys.forEach((key, idx) => {
    const block = SERVICE_PDF_CONTENT[key];
    if (!block) return;
    
    // Section header
    y = ensurePdfSpace(doc, y, 38);  // Check page overflow
    sectionHeader(doc, `  INFORMACION RELEVANTE Y EXTRA · ${block.title.toUpperCase()}`, margin, y, contentWidth);
    y += 8;
    
    // "Información relevante" paragraphs
    doc.setFont('helvetica', 'bold');
    doc.text('Información relevante:', margin, y);
    y += 4;
    
    doc.setFont('helvetica', 'normal');
    block.relevant.forEach(paragraph => {
      const lines = doc.splitTextToSize(paragraph, contentWidth);
      y = ensurePdfSpace(doc, y, lines.length * 4 + 2);
      doc.text(lines, margin, y);
      y += lines.length * 4 + 1;
    });
    
    // "Información extra" bullet points
    y += 2;
    doc.setFont('helvetica', 'bold');
    doc.text('Información extra:', margin, y);
    y += 4;
    
    doc.setFont('helvetica', 'normal');
    block.extra.forEach(item => {
      const lines = doc.splitTextToSize(`- ${item}`, contentWidth);
      y = ensurePdfSpace(doc, y, lines.length * 4 + 1);
      doc.text(lines, margin, y);
      y += lines.length * 4 + 1;
    });
    
    if (idx < keys.length - 1) y += 5;  // Spacing between services
  });
  
  return y;
}
```

<Note>
  The "cambio" service automatically includes "instalacion" terms since it involves equipment replacement plus installation.
</Note>

## Logo Customization

The company logo is embedded as a base64 PNG (app.js:1386-1418):

### Logo Configuration

```javascript
const PDF_LOGO_BASE64 = 'iVBORw0KGgoAAAANSUhEUgAAB9AAAAZuCAYAAAAcj73IAAEAAElEQVR42uz9...';

function normalizePngDataUrl(input) {
  if (!input) return null;
  const raw = String(input).trim();
  
  if (!raw) return null;
  
  // Accept full data URL or raw base64
  if (raw.startsWith('data:image/png;base64,')) {
    return raw;
  }
  
  // Clean whitespace from multiline base64
  const cleanBase64 = raw.replace(/\s+/g, '');
  return `data:image/png;base64,${cleanBase64}`;
}

async function getPdfLogoSource() {
  if (pdfLogoCache) return pdfLogoCache;
  
  const dataUrl = normalizePngDataUrl(PDF_LOGO_BASE64);
  if (!dataUrl) return null;
  
  pdfLogoCache = dataUrl;
  return dataUrl;
}
```

## Page Overflow Handling

The system automatically adds pages when content exceeds available space (app.js:1429-1433):

```javascript
function ensurePdfSpace(doc, y, neededHeight, pageH = 297) {
  if (y + neededHeight <= pageH - 22) {  // 22mm reserved for footer
    return y;
  }
  
  doc.addPage();  // Start new page
  return 20;      // Reset Y position with top margin
}
```

Used before each major section:

```javascript
y = ensurePdfSpace(doc, y, 38);  // Check before section header
sectionHeader(doc, 'SECTION TITLE', margin, y, width);
```

## Error Handling

PDF generation includes validation:

```javascript
function validateQuoteForPDF() {
  const clientName = document.getElementById('clientName').value;
  if (!clientName || !clientName.trim()) {
    showToast('Ingresa el nombre del cliente', 'error');
    goToStep(4);
    return false;
  }
  
  const clientAddress = document.getElementById('clientAddress').value;
  if (!clientAddress || !clientAddress.trim()) {
    showToast('Ingresa la dirección de instalación', 'error');
    goToStep(4);
    return false;
  }
  
  return true;
}
```

Call before generating:

```javascript
async function generatePDF() {
  if (!validateQuoteForPDF()) return;
  
  // ... proceed with PDF generation
}
```

## Performance

### Generation Time

Typical PDFs generate in:
- Simple quote (1 equipment, 1 service): ~200ms
- Complex quote (5+ items, multiple services, notes): ~500ms
- With logo: +100ms (first generation only, then cached)

### Optimization Tips

<Accordion title="For Large Tables">
  - autoTable handles pagination automatically
  - No manual optimization needed for up to 50 rows
  - Beyond 50 rows, consider summarizing accessories
</Accordion>

<Accordion title="For Logo">
  - Cache logo data URL after first load
  - Use PNG (faster than SVG in jsPDF)
  - Keep logo file under 100KB
  - Pre-compress images before base64 encoding
</Accordion>

<Accordion title="For Text Wrapping">
  - `splitTextToSize()` is efficient for normal paragraphs
  - Pre-calculate line counts for complex layouts
  - Avoid excessive calls in tight loops
</Accordion>
